# 스터디 허브

논문 스터디, 운동, 독서 스터디, 연구 등 팀 스터디·활동을 한곳에서 관리하는 웹앱입니다.

## 팀원

- 창민석, 이윤지, 송수현, 강태영, 조수민, 신현호

## 기능

- **팀원**: 멤버 목록 한눈에 보기
- **스터디 & 활동**: 논문 스터디, 운동, 독서 스터디, 연구 카드
- **다가오는 일정**: 일정 추가·삭제 (브라우저 로컬 저장)

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기 (더블클릭 또는 드래그)
2. 또는 로컬 서버로 실행:
   ```bash
   cd study-hub
   npx serve .
   ```
   실행 후 브라우저에서 표시되는 주소(예: http://localhost:3000)로 접속

일정은 이 기기 브라우저에만 저장됩니다. 다른 기기/브라우저에서는 새로 등록해야 합니다.
